Clinical Neurophenomenology Interface

NeuroPhenom AI

A research-facing environment for live micro-phenomenological interviewing, structured qualitative synthesis, and privacy-conscious experiential analysis.

Open Live App View Repository

What It Does

  • Runs structured neurophenomenology interviews with configurable tone and sensitivity.
  • Generates summaries, takeaways, diachronic structure, synchronic structure, and coding suggestions.
  • Supports live interviewing workflows alongside post-session qualitative analysis.
  • Targets a privacy-first clinical-research posture rather than a generic chatbot experience.
